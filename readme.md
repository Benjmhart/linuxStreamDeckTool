# Stream Deck Linux Daemon

this is a basic system to create automation with Streamdeck in Linux. currently, you need to clone the repo, manually fill in ./streamDeckrc.js, and add any graphics to ./images.

## usage

    npm run start

...on some systems

    sudo npm run start
